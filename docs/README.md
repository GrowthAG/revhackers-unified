# 🚀 RevHackers B2B Growth Hub

Este é o repositório proprietário do núcleo de operações tecnológicas da **RevHackers**, uma assessoria focada no crescimento corporativo de alto ticket e estruturação de Máquinas de Vendas (RevOps).

## Arquitetura Tecnológica
Esta aplicação foi projetada para lidar com operações pesadas de diagnósticos B2B, geração de propostas dinâmicas em tempo real, geração e gerenciamento e assinaturas eletrônicas.

**Tech Stack Primária:**
- React (Vite)
- Tailwind CSS & Shadcn-UI
- Supabase (PostgreSQL, Auth, Storage)
- Puppeteer (Pré-renderização de SEO dinâmico)
- Framer Motion (Micro-interações de CRO)

## Módulos Principais
1. **Frontend (B2C/B2B Leitura):** Páginas estáticas rápidas, blog otimizado para Core Web Vitals e Capturas Automáticas Inteligentes (CTAs Baseados no Conteúdo).
2. **Growth Hub (Client Area):** Área restrita com `Deal Rooms` para clientes monitorarem implementações, Sprints Semanais e ROI.
3. **Legal Vault (Cofre Legal):** Assinatura eletrônica de metadados focada em Proposals/Cotações criptografadas (SHA-256).

## Inicialização e Deploy
Para rodar este HUB localmente:

\`\`\`bash
npm install
npm run dev
\`\`\`
