# RevHackers Glass Artifacts - Master Prompt Template

Use este template para gerar capas de blog consistentes com a identidade visual "Glass Neon Engineering".

## 📋 The Master Prompt

> **Copy and paste this into the image generator (Imagen 3 / Midjourney):**

```text
A wide 16:9 cinematic shot of a minimalist 3D glass icon representing [TOPIC]. A sleek, translucent dark glass [OBJECT] floating in a vast deep black void. Inside the glass, glowing neon green [INTERNAL_DETAILS]. Premium glassmorphism, sharp edges, internal light reflection, soft neon green rim lighting. IMPORTANT: The object is SMALL in the frame (approx 25-30% height), perfectly centered, with massive negative space around it. Identical scale (zoom out) to a minimalist icon. No text. 8k resolution. Pure engineering artifact style.
```

---

## 🧩 Variables to Fill

1.  **[TOPIC]**: O tema do artigo (ex: "Product Market Fit", "Sales Commission", "Automation").
2.  **[OBJECT]**: O objeto físico que será feito de vidro. Deve ser simples e icônico.
    *   *Exemplos:* "Puzzle piece", "Robot hand", "Balance scale", "Hourglass".
3.  **[INTERNAL_DETAILS]**: O que brilha dentro do vidro (Neon).
    *   *Exemplos:* "glowing data circuits", "continuous neon flow", "binary code matrix", "energy core".

---

## 🎨 Design Rules (Critical)

1.  **Scale (Escala):** O objeto DEVE ser **PEQUENO** (Small/Distant). Se ficar grande, quebra o padrão. O prompt contém instruções explícitas ("SMALL", "approx 30% height") para garantir isso.
2.  **Color Palette:**
    *   **Glass:** Dark Grey / Translucent Black.
    *   **Light/Neon:** RevGreen (Neon Green).
    *   **Background:** Deep Void Black (#000000).
3.  **Composition:** Sempre **Centralizado** com muito espaço negativo (preto) em volta. Isso permite que a imagem seja cortada em qualquer formato (celular, card vertical) sem perder o objeto.
